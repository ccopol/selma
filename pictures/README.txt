How to export repeat*.svg to png
================================
1) Use Inkscape
2) Select the area to export
3) File -> Export Bitmap...
4) Make sure that the size of the export area is the same for all exports (x1: 128; x2=276; y1=757; y2=822), otherwise in the app it will seem as if the icon is jumping
5) Make sure to select the same bitmap size (x=32; y=14), otherwise in the app it will seem as if the icon is jumping
